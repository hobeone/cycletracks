WTF is this?

Cycletracks is a webapp to store and view information from a Garmin cycle computer (I've developed it using my Garmin 705). I put it together after getting fed up with the really really long load times of Motionbased and Connect. I've since added support for more generic gps information by supporting GPX files.

If you want to help develop the app please let me know. You can grab the source from the link below and get hacking.

What GPS devices does it support?

For TCX files:

I only own the Garmin Edge 705 so that is the only device I know works with it. The 605 should work as the are basically the same. I don't know if the 205 or 305 will work. If you have one you will need to extract the data into a TCX formatted data file. If you figure out how to do this drop me a line in the discussion group.

For GPX files:

The GPX support should allow any device that can export a GPX file to upload to cycletracks. GPX doesn't support all the data that TCX does so some information will be missing (Cadence, Heart Rate) and some data is calculated by cycletracks so may be less accurate (distance and speed). If your device doesn't export GPX but some other format you may be able to use a program like GPSBabel to convert it to GPX.

Setting up your account:

  • Click on the 'Settings' Tab and choose your timezone and what kind of units you would like to display (Metric or Imperial).

What you can do:

  • Upload your TCX and GPX files (either one at a time or as a compressed archive).
  • See the grand totals of all your activities.
  • See a detailed view of one activity with a Google map, and graphs for heart rate, cadence etc.
  • Download a kml file of an activity for viewing in Google Earth or Google Maps
  • Send a link to an activity to your firends.

How do I share an activity with my friends?

  • Go to the detailed view of the activity.
  • Click on the 'False' text next to the 'Public' text in the top left summary table
  • It should turn into a drop down selection box. Change 'False' to 'True' and click somewhere else on the page.
  • Once that change has been saved you can send a link to the activity to others and they will be able to see it. If you use the public link (right below the Google map) they wont need to be logged in to Google to see the activity
Caveats about data representation in cycletracks:
  • Times are rounded to integers of seconds
  • Map data is encoded before rendering (to save space) but this may introduce some imprecision into the geo points as shown on the Google maps.
Other Notes:
  • To change the name of an activity click on the name in the show screen, an edit box should appear.
  • This should work with the public, comment and tag fields too
